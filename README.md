# gorkor-cli

>一个简单的脚手架工具


### Installation

```
 npm install -g gorkor-cli
```


### Usage

```
gk init
```

##### Example
>使用默认的gk-cli

需要构建模版名称: gk-cli  
生成的项目名称: demo

```
 cd demo && npm install
```
